Add your own games

Put each HTML5 game in its own folder under games/, then add a card for it in external-games.js. Use only games you made, open-source games with a license that allows reuse, or embeds the creator specifically allows.

  1. Create games/my-game/index.html.
  2. Add assets beside it using relative paths.
  3. Add a game card entry in external-games.js.
  4. Credit the creator if the license requires it.
